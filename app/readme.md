## 服务编码:

- ✅ 服务的端口号 = 服务编号 + 服务类型编码

| 服务类型编码 | 说明        | 重要性  |
|--------|-----------|------|
| 0      | rpc       | ⭐⭐⭐️ |
| 1      | job       | ⭐    |
| 2      | admin api | ⭐    |
| 8      | web api   | ⭐⭐⭐  |
